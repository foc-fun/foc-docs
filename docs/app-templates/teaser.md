## Preview

Here's a glimpse of how easy it will be to use App Templates:

```bash
# Install foc CLI
npm install -g @foc/cli

# Create new project from template
foc create my-dex --template defi/dex

# Configure your project
cd my-dex
foc configure

# Deploy to testnet
foc deploy --network testnet

# Launch locally
foc dev
```
